Introduction & Project Overview

The Library Management System (LMS) is a full-stack web application designed to automate and digitize all daily operations of a college library. It eliminates manual paperwork and provides a fast, secure, and user-friendly platform for managing books, members, and transactions.

This system supports two user roles — Admin (librarian) and Member (student). Admins can manage the entire library catalog, issue or return books, track overdue fines, and generate reports. Members can log in to view their issued books and profile.

The project was developed using Python with the Flask micro web framework, SQLite as the embedded database, and HTML, CSS, JavaScript for the responsive dark-themed frontend.

Database Design & Tables

The system uses a SQLite relational database named library.db with 3 main tables: users, books, and transactions. The database is automatically created and seeded with sample data on the first app run.

Relationships: The transactions table links to both books and users via foreign keys — one transaction belongs to one book and one member.

Key Features of the System

Role-Based Login

Admin & Member roles with hashed passwords and secure Flask sessions.

Book Catalog (CRUD)

Add, edit, delete, and search books by title, author, ISBN, or category.

Member Management

Register members, activate/deactivate accounts, and view issue history.

Issue & Return

Issue books with a 14-day due date; return and auto-calculate ₹5/day fine.

Reports & Analytics

Monthly charts, most-borrowed books, category stats, and overdue lists.

Mobile Responsive

Fully responsive UI for desktop, tablet, and mobile with collapsible sidebar.

How to Run the Project
1
Install Python 3

Download from python.org and install Python 3.10 or above on your computer.

2
Open Terminal in Project Folder

Open Command Prompt or PowerShell and navigate to the library management system folder.

3
Install Dependencies

Run: pip install flask werkzeug to install all required packages.

4
Start the App

Run: python app.py — the database will be auto-created and the server will start.

Open in Browser
