Library Management System

A full-stack web application for managing library books, members, and transactions — built with Python, Flask, SQLite, HTML, CSS & JavaScript.

Python Flask SQLite HTML/CSS/JS

Project Overview

The Library Management System (LMS) is a web-based application designed to automate and streamline all day-to-day operations of a library. It replaces manual record-keeping with a secure, fast, and easy-to-use digital platform.

Technologies & Languages Used

Python 3 Backend Language

Python is the core programming language used for all backend logic. It is simple, powerful, and widely used for web development.

  • Handles all server-side logic
  • Processes form data and database queries
  • Manages user sessions and authentication
  • Calculates overdue fines automatically
Learn Python — Programming with Mosh
Flask Web Framework

Flask is a lightweight Python web framework used to build the web routes, handle HTTP requests, and render HTML templates.

  • Defines all URL routes (login, books, members, etc.)
  • Renders Jinja2 HTML templates
  • Manages login sessions with flask-login
  • Handles POST/GET form submissions securely
Learn Flask — freeCodeCamp
SQLite Database

SQLite is an embedded relational database stored as a single file (library.db). No separate server is required — it works out of the box.

  • Stores all books, members and transactions
  • Supports SQL queries for fast data retrieval
  • Lightweight — no installation needed
  • Automatically created on first run
Learn SQLite — TechWithTim
HTML / CSS / JavaScript Frontend

The frontend is built with pure HTML5, CSS3, and Vanilla JavaScript for a modern, responsive, dark-themed user interface.

  • HTML (Jinja2) — page structure and templates
  • CSS — premium dark theme, animations, mobile layout
  • JavaScript — sidebar toggle, chart animations, auto-search
  • Font Awesome — icons throughout the app
Learn HTML/CSS/JS — freeCodeCamp

Key Features of the System

Role-Based Login

Two roles: Admin (full access) and Member (view only). Passwords are hashed using Werkzeug for security.

Book Catalog

Full CRUD operations — Add, Edit, Delete, and Search books by title, author, ISBN, or category.

Member Management

Register members with email/password, activate or deactivate accounts, and view issue history.

Issue & Return

Issue books with automatic 14-day due dates. Return books and auto-calculate overdue fines at ₹5/day.

Reports & Analytics

Monthly issue charts, most borrowed books, category statistics, and complete overdue book reports.

Mobile Responsive

Fully responsive layout that works on desktops, tablets, and smartphones with a collapsible sidebar.

How to Run the Project

1
Install Python

Download Python 3.10+ from python.org and install it.

2
Install Dependencies

Open terminal in the project folder and run:
pip install flask werkzeug

3
Run the App

In the same terminal run:
python app.py

4
Open in Browser

Visit http://127.0.0.1:5000 and log in.
